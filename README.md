# Сайт для разметки видеофайлов на MTurk

## Быстрый старт

Установка зависимостей Node.js (выполняется 1 раз).
```sh
npm i
```

Запуск сервера.
```sh
npm start
```

Разработка бекенда.
```sh
nodemon app.js
```

Разработка фронтенда.
```sh
ng serve
```

После установки и запуска сайт становится доступным по адресу `http://localhost:4200/`.
