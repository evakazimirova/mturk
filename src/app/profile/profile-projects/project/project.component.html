При нажатии на название проекта открывается страница проекта, где заказчик описывает объем работы, предоставляет описание как работать с контентом, прикрепляет картинки и видео с необходимыми результатами, или видео с процессом выполнения разметки.

Процент выполнения. Заказчик определяет минимальный объем работы для аннотатора, например, разметка 20 видео по 3 шкалам это один оплачиваемый блок. После окончания можно взять еще один блок.

Срок сдачи. Если заказчик определяет минимальный объем работы, и срок, то он указывается правее статуса. В сроке указано время и часовой пояс.
